Comentarios

En PHP, los comentarios son fragmentos de texto dentro del código que no se ejecutan. Sirven para dejar notas, aclaraciones o explicaciones para ti mismo u otras personas que lean el programa. Son muy útiles para: Explicar qué hace una parte del código. Recordar por qué escribiste algo de cierta manera. Desactivar temporalmente una línea de código sin borrarla.

// Comentario de una línea
# Comentario estilo shell
/* Comentario de
   varias líneas */

Concatenación

En **PHP**, la **concatenación** es la operación que permite **unir dos o más cadenas de texto (strings)** en una sola. Se utiliza principalmente para construir mensajes, combinar variables con texto o formar valores dinámicos. En PHP, la concatenación se hace con el **operador punto (`.`)**. También existe el operador de **asignación con concatenación (`.=`)** que añade contenido a una cadena ya existente. En resumen: la concatenación sirve para **juntar cadenas y variables en un solo string** dentro de un programa en PHP. 

$nombre = "Juan";
$saludo = "Hola " . $nombre;
echo $saludo; // Hola Juan

Variables

En PHP, una variable es un espacio de memoria que se utiliza para almacenar datos que pueden cambiar durante la ejecución de un programa. Las características principales de las variables en PHP son: Siempre comienzan con el signo de dólar ($). El nombre distingue entre mayúsculas y minúsculas. No es necesario declarar el tipo de dato, ya que PHP es un lenguaje de tipado débil (la variable puede cambiar de tipo automáticamente según el valor asignado). Pueden almacenar diferentes tipos de datos, como números, cadenas de texto, arreglos u objetos. En resumen: una variable en PHP es un contenedor dinámico de información que se puede usar y modificar en el programa.

$nombre = "Daniel";
$edad = 20;
$esAlumno = true;

Expresiones

En **PHP**, una **expresión** es cualquier combinación válida de **valores, variables, operadores y funciones** que produce un **resultado**. Algunas características: * Siempre generan un valor, aunque sea temporal. * Pueden ser simples (un número, una cadena, una variable) o complejas (operaciones aritméticas, lógicas o llamadas a funciones). * Son los componentes básicos que PHP evalúa dentro de sentencias y estructuras de control. En resumen: una expresión en PHP es una **unidad de código que PHP evalúa y resuelve en un valor**. 

$expresion = (3 + 5) * 2; // 16

Operadores

En **PHP**, los **operadores** son elementos del lenguaje que se utilizan para **realizar distintas operaciones sobre los datos**, ya sean valores literales, variables o resultados de expresiones. Permiten manipular, comparar y combinar información dentro del programa, y son fundamentales para controlar la lógica, los cálculos y la interacción entre los diferentes elementos del código. En otras palabras, los operadores son la **herramienta principal que PHP ofrece para procesar y transformar datos de manera dinámica**.

Precedencia de operadores

La **precedencia de operadores** en PHP se refiere al **orden en que se ejecutan las operaciones** cuando una expresión contiene varios operadores diferentes. Esto determina **qué operación se realiza primero**, incluso sin paréntesis. Si no se respeta este orden, el resultado de la expresión puede ser diferente al esperado. En resumen: la precedencia de operadores establece **la prioridad de ejecución entre los distintos operadores** dentro de una expresión en PHP. 

$resultado = 5 + 3 * 2; // 11 (multiplicación antes que suma)

Operadores aritméticos

Los **operadores aritméticos** en PHP son **símbolos que permiten realizar operaciones matemáticas básicas** sobre números, como suma, resta, multiplicación, división y módulo. Se utilizan para **calcular valores numéricos, manipular variables y realizar operaciones matemáticas dentro de un programa**. 

+  Suma
-  Resta
*  Multiplicación
/  División
%  Módulo (residuo)

Operadores de asignación

Los **operadores de asignación** en PHP son herramientas que permiten **asignar valores a las variables** y, al mismo tiempo, **modificar su contenido de manera eficiente**. El operador más básico es el de asignación simple (`=`), que toma el valor de la derecha y lo almacena en la variable de la izquierda. Además, existen operadores de asignación combinados que **fusionan una operación con la asignación**, como sumar, restar, multiplicar o dividir directamente sobre el valor de la variable sin necesidad de escribir la operación completa por separado. Estos operadores son fundamentales porque **facilitan la manipulación de datos numéricos, cadenas y otros tipos de variables**, haciendo el código más compacto y legible. También ayudan a **evitar errores al actualizar valores repetidamente** dentro de programas más complejos. 

$x = 10;
$x += 5; // 15
$x -= 3; // 12

Operadores de comparación

Los **operadores de comparación** en PHP son símbolos que se utilizan para **comparar dos valores** y determinar la relación entre ellos. El resultado de una comparación siempre es **un valor booleano**: verdadero (`true`) o falso (`false`). Se usan para **tomar decisiones en el programa**, como decidir qué bloque de código ejecutar dentro de estructuras de control (`if`, `while`, etc.) según si una condición se cumple o no. 

== Igual
!= Diferente
>  Mayor
<  Menor
>= Mayor o igual
<= Menor o igual
=== Igual en valor y tipo

Operadores de incremento/decremento

Los **operadores de incremento y decremento** en PHP son símbolos que se utilizan para **aumentar o disminuir el valor de una variable en una unidad**. Permiten **modificar rápidamente el valor de una variable numérica** sin tener que escribir la operación completa. Son muy útiles en **bucles y ciclos**, donde se necesita ajustar contadores o índices de manera eficiente. 

$x = 5;
echo ++$x; // 6 (pre-incremento)
echo $x--; // 6, pero luego $x=5

Operadores lógicos

Los **operadores lógicos** en PHP son símbolos que se utilizan para **combinar o invertir expresiones booleanas**. Su función principal es **evaluar condiciones compuestas**, permitiendo que el programa decida el flujo de ejecución según múltiples criterios. Los resultados de estos operadores siempre son **verdadero (`true`) o falso (`false`)**, y son fundamentales en estructuras de control como `if`, `while` o `for`. 

&&  AND (y)
||  OR (o)
!   NOT (no)
      
$edad = 20;
if ($edad > 18 && $edad < 30) {
  echo "Edad válida";
}

Operadores para strings

Los **operadores para strings** en PHP son herramientas que permiten **trabajar de manera flexible con cadenas de texto**, realizando operaciones como unir varias cadenas, agregar contenido a una cadena existente o comparar textos. Estos operadores son fundamentales cuando se necesita **generar mensajes dinámicos, construir resultados basados en variables o procesar datos textuales**. Facilitan la manipulación de strings sin necesidad de recurrir a funciones más complejas, haciendo el código más **claro y eficiente**.

Operadores para arrays

Los **operadores para arrays** en PHP son símbolos que se utilizan para **manipular y comparar arreglos (arrays)** dentro de un programa. Permiten **unir, comparar o verificar la igualdad de arrays**, facilitando la gestión de colecciones de datos, la combinación de múltiples arrays y la evaluación de su contenido de manera lógica y estructurada. Estos operadores ayudan a **simplificar operaciones complejas sobre conjuntos de datos** sin necesidad de recorrerlos manualmente. 

$a = [1,2];
$b = [2,3];
$c = $a + $b; // Unión
$a == $b;     // Igualdad de valores
$a === $b;    // Igualdad de valores y tipos

Operadores de tipo

Los **operadores de tipo** en PHP son símbolos que se utilizan para **verificar o convertir el tipo de dato de una variable**. Permiten determinar si una variable es de un tipo específico, como entero, cadena, arreglo u objeto, y también facilitan **castear o transformar valores de un tipo a otro** cuando es necesario para la ejecución correcta del programa. Estos operadores son importantes para **controlar y garantizar la coherencia de los datos** en las operaciones y evitar errores por tipos incompatibles. .

var_dump(is_int(5)); // true
var_dump(is_string("hola")); // true

Estructuras de Control

Las **estructuras de control** en PHP son **mecanismos que permiten dirigir el flujo de ejecución del programa** según ciertas condiciones o repeticiones. Se utilizan para **tomar decisiones, ejecutar código de manera condicional o repetir acciones múltiples veces**, dependiendo de los valores de variables o el cumplimiento de expresiones lógicas. En otras palabras, son herramientas que **controlan qué instrucciones se ejecutan, cuándo y cuántas veces**, haciendo que el programa sea dinámico y adaptable. if, else, while, for, switch, etc.

Sentencia if

La **sentencia `if`** en PHP es una **estructura de control que permite ejecutar un bloque de código solo si se cumple una condición determinada**. Se utiliza para **tomar decisiones dentro del programa**, evaluando expresiones booleanas y permitiendo que ciertas instrucciones se ejecuten únicamente cuando la condición es verdadera. Esto hace que el flujo del programa sea **condicional y flexible**, adaptándose a distintas situaciones según los datos disponibles. 

$edad = 18;

if ($edad >= 18) {
  echo "Eres mayor de edad";
} else {
  echo "Eres menor de edad";
}
