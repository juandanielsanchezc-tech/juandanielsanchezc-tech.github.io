Ordenar un arreglo bidimensional

Un arreglo bidimensional es una matriz (tabla) compuesta por filas y columnas que almacena datos de forma estructurada. Para ordenarlo existen varias estrategias: puedes recorrer todos sus elementos y aplicar un algoritmo de ordenamiento (como burbuja, selección o inserción) tratándolo como si fuera una sola lista —aplanando la matriz—, o bien aplicar el ordenamiento por cada fila o cada columna, según lo que necesites resolver. Esto permite organizar la información de manera más clara y facilitar operaciones posteriores, como búsquedas o cálculos.

Ejemplo completo (aplanar, ordenar y reconstruir usando for):

<?php
if (isset($_POST['ordenar'])) {
    $n = $_POST['numeros'];

    for ($i = 0; $i < 3; $i++) {
        for ($j = 0; $j < 3; $j++) {
            for ($x = $i; $x < 3; $x++) {
                for ($y = ($x == $i ? $j : 0); $y < 3; $y++) {
                    if ($n[$i][$j] > $n[$x][$y]) {
                        $aux = $n[$i][$j];
                        $n[$i][$j] = $n[$x][$y];
                        $n[$x][$y] = $aux;
                    }
                }
            }
        }
    }

    echo "
Arreglo ordenado
";
   
    for ($i = 0; $i < 3; $i++) {
        echo "";
        for ($j = 0; $j < 3; $j++) {
            echo "" . $n[$i][$j] . "";
        }
        echo "";
    }
    echo "";
}
?>>

Suma de matrices

La suma de dos matrices solo es posible si ambas tienen las mismas dimensiones (igual número de filas y columnas). El procedimiento consiste en sumar cada elemento de la primera matriz con el que se encuentra en la misma posición (i, j) de la segunda. El resultado es una nueva matriz de las mismas dimensiones donde cada valor representa la suma correspondiente. Este proceso es muy usado en álgebra lineal y en programación para operaciones con datos estructurados.

Ejemplo:

A = [1 2]   B = [4 5]
    [3 4]       [6 7]

A + B = [5 7]

Multiplicación de matrices

La multiplicación de matrices A × B solo es posible si el número de columnas de la primera matriz (A) coincide con el número de filas de la segunda (B). El resultado será una nueva matriz donde cada elemento (i, j) se obtiene calculando la suma de los productos entre los elementos de la fila i de A y los elementos de la columna j de B. Este procedimiento es fundamental en matemáticas aplicadas, álgebra lineal, gráficos por computadora y programación, ya que permite combinar transformaciones y representar relaciones entre conjuntos de datos.

Ejemplo:

A = [1 2]   B = [5 6]
    [3 4]       [7 8]

A x B = [
 (1*5+2*7) (1*6+2*8)
 (3*5+4*7) (3*6+4*8)
] = [19 22][43 50]

Prevención de Adicciones — Día Mundial de la Salud Sexual

El enfoque en la salud sexual incluye promover prácticas seguras, consentidas e informadas. Hablar abiertamente sobre sexualidad responsable ayuda a reducir mitos y estigmas, fomentando la confianza en jóvenes para tomar decisiones conscientes sobre su bienestar.

Las adicciones (alcohol, drogas, etc.) aumentan conductas de riesgo sexual y disminuyen la toma de decisiones seguras. El consumo de estas sustancias puede provocar falta de protección en las relaciones sexuales, mayor exposición a infecciones de transmisión sexual (ITS), embarazos no planeados y violencia.

Mensajes clave: educación sexual integral, acceso a servicios de salud, prevención de consumo y apoyo emocional para jóvenes. Relacionar campañas de salud sexual con prevención de adicciones fortalece el cuidado integral, creando espacios de confianza donde los adolescentes puedan resolver dudas y expresar inquietudes.

La combinación de programas escolares, orientación familiar y acceso a servicios de salud accesibles permite un acompañamiento más sólido. Además, promover actividades recreativas, deportivas y culturales se convierte en una estrategia preventiva que ofrece alternativas positivas frente al consumo de sustancias.

Prevención de Adicciones — Fentanilo

El fentanilo es un opioide sintético extremadamente potente, hasta 50 veces más fuerte que la heroína y 100 veces más que la morfina. Su uso médico controlado ayuda en casos de dolor intenso, pero el consumo ilegal (mezclado con otras drogas) ha causado un aumento dramático de sobredosis, ya que pequeñas cantidades pueden ser mortales.

Puntos importantes de prevención: promover la educación sobre los riesgos, evitar cualquier consumo no médico, reconocer señales de sobredosis (respiración lenta, pupilas muy pequeñas, inconsciencia) y buscar ayuda inmediata llamando a emergencias o acudiendo a servicios de salud especializados. También es clave fomentar el apoyo familiar y comunitario, además de impulsar campañas de concientización en jóvenes y adultos.
