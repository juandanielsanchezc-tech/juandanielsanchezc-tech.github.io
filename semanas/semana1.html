¿Qué es PHP?

PHP (PHP: Hypertext Preprocessor) es un lenguaje de programación de código abierto, diseñado especialmente para el desarrollo de aplicaciones web. Se integra fácilmente con HTML y permite crear páginas dinámicas.

A diferencia de JavaScript, PHP se ejecuta en el servidor. El cliente solo recibe el resultado en forma de HTML, sin acceso al código que lo generó. Esto lo hace seguro y potente para aplicaciones web.

Imagen de referencia: PHP

¿Qué puede hacer PHP?

PHP puede hacer prácticamente todo: procesar formularios, generar contenido dinámico, manejar sesiones, gestionar cookies y comunicarse con bases de datos.

Además, puede ejecutarse desde la línea de comandos para realizar tareas programadas o manipular archivos de texto, lo que lo convierte en una herramienta muy versátil.

Instalar MAMP

  • Ingresar a la página oficial: Descargar MAMP.
  • Seleccionar el instalador según tu sistema operativo (Windows o macOS).
  • Ejecutar el instalador:
    • En Windows: MAMP_Setup.exe.
    • En macOS: MAMP.pkg.
  • Aceptar los términos y condiciones.
  • Elegir carpeta de instalación:
    • Windows: C:\MAMP (recomendado).
    • macOS: se instala en Aplicaciones.
  • Finalizar la instalación y abrir la aplicación MAMP.
  • Iniciar los servidores haciendo clic en Start Servers.
  • Guardar tus archivos en la carpeta de trabajo: htdocs.
  • Comprobar la instalación en el navegador: http://localhost.

Visualizar archivos

Para visualizar un archivo, abre el navegador y escribe en la barra de direcciones: http://localhost/archivo.php. Si configuraste un puerto distinto, añade el número: http://localhost:8888/archivo.php.

Variables en PHP

En PHP las variables comienzan con un signo de dólar $ seguido por su nombre. Los nombres son sensibles a mayúsculas y minúsculas. Deben empezar con una letra o guión bajo, seguidos de letras, números o guiones bajos.
